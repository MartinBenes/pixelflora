# Support

## Supported versions

| Version line | Support level  |
| ------------ | -------------- |
| 1.x          | Active support |
| < 1.0        | Not supported  |

## How to get help

- Usage questions and bug reports: open a GitHub issue using templates in `.github/ISSUE_TEMPLATE`.
- Security vulnerabilities: follow private reporting instructions in `SECURITY.md`.

## Response targets

- New issue triage: within 5 business days.
- Security triage: within 3 business days (see `SECURITY.md`).

Response targets are goals, not legal guarantees.
